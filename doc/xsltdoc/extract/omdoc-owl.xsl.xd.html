krextor:make-catalogue (param: node()* todo, node()* theory, base-uri, node()* visited, boolean top-level-call) - source
Recursively examine the locally imported theories and locate all theories that still need a catalogue entry
Each theory is only visited once. Returns a set of elements.
Parameters:
node()* todo - the theories that still need a catalogue entry
node()* theory - the theory that is searched for them
base-uri - the base URI of the current theory, against which links to imported theories are resolved
node()* visited - a set of theories already visited
boolean top-level-call - indicates whether this is a top-level or a recursive call
