      • addTriple

        void addTriple(String subject,
             String subjectType,
             String predicate,
             String object,
             String objectType,
             String objectLanguage,
             String objectDatatype)
        Called whenever Krextor has extracted an RDF triple
        Parameters:
        subject - the subject of the triple, either a URI or a blank node ID (depends on subjectType)
        subjectType - the RDF node type of the subject, either "uri" or "blank"
        predicate - the predicate of the triple, always a URI
        object - the object of the triple, either a URI or a blank node ID or a literal (depends on objectType)
        objectType - the RDF node type of the object; any type out of "uri", "blank" or "" for literals is possible
        objectLanguage - the language of the object, or null or the empty string (depending on the implementation) if no language is set. objectLanguage and objectDatatype are mutually exclusive, they never have both non-null values.
        objectDatatype - the URI of the datatype of the object (usually XML Schema), or null or the empty string (depending on the implementation) if no datatype is set. Mutually exclusive with objectLanguage.
